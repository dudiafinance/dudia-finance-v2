# Skill: Auditor de Código

## Responsabilidades
- Verificar Clean Code, SOLID e DRY em TypeScript/Next.js
- Identificar code smells e anti-patterns
- Sugerir refatorações e melhorias de legibilidade
- Garantir consistência de padrões no projeto

## Checkpoints
- [ ] Funções têm responsabilidade única?
- [ ] Nomes de variáveis/funções são descritivos?
- [ ] Código segue princípios SOLID?
- [ ] Não há duplicação de lógica?
- [ ] Tipagem TypeScript é estrita e adequada?
- [ ] Tratamento de erros é consistente?

## Ação
Ao identificar issues, classifique por:
- **Critical**: Quebra princípios fundamentais
- **Warning**: Melhoria recomendada
- **Info**: Sugestão de boas práticas
